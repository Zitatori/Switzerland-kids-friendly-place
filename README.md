## What this app does
Personal database of kids-friendly places in Switzerland.
Deploy Link:
https://zitatori-switzerland-kids-friendly-place-streamlit-app-jiwb1k.streamlit.app/

## Key features
- Add new place (form)
- Browse places (cards UI)
- Filter by category / location
- Mobile-friendly

## Tech stack
- Python
- Streamlit
- Flask (registration)
- Supabase (database)

## Why I built this
To manage real-life family outings efficiently.

## Future improvements
- Map integration
- Rating system
- Image upload
